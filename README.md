# 社交媒体研究工具包

一个使用 Claude Code 和 TikHub API 进行深度网络研究和社交媒体数据收集的综合研究工具包。该工具包通过并行多智能体研究方法，能够对社交媒体影响者、趋势和跨文化影响进行全面分析。

## 概述

该仓库包含一个复杂的社交媒体研究系统，将 Claude Code 的智能体技能与用于深度研究和社交媒体数据收集的专用工具相结合。该工具包已被用于生成关于李子柒和姚明等知名人士的详细研究报告，展示了其进行全面社交媒体分析的能力。

## 功能特性

- **并行研究智能体**: 同时部署多个子智能体以实现更快、更全面的研究
- **多平台社交媒体 API 访问**: 集成 TikHub API 以访问 TikTok、抖音、小红书、Instagram、YouTube、Twitter、Reddit、Bilibili、微博和知乎
- **自适应工具选择**: 为不同类型的内容自动选择最佳工具（网络搜索、内容提取、JavaScript 渲染）
- **结构化研究报告**: 生成包含定量数据和定性分析的全面、有引用的研究报告
- **跨文化分析**: 专门用于分析不同文化背景下的社交媒体影响

## 架构

该工具包由四个主要技能组成：

### 1. 深度研究主智能体 (`deep-research`)
- 协调综合研究流程
- 分类查询并确定最佳研究策略
- 部署并行子智能体以提高效率
- 将发现整合到最终报告中

### 2. 研究子智能体 (`research-subagent`)
- 执行主智能体委托的集中研究任务
- 高效使用工具，最多 20 次工具调用
- 采用 OODA（观察、定位、决策、行动）方法论

### 3. TikHub API 助手 (`tikhub-api-helper`)
- 搜索和查询 TikHub API 以获取社交媒体数据
- 支持 14+ 个社交媒体平台，数千个 API 端点
- 包含 API 发现和客户端请求工具

### 4. 引用智能体 (`citations`)
- 为研究报告添加适当的引用
- 使用脚注或内联引用格式
- 维护来源完整性和可信度

## 使用方法

### 启动研究任务

要启动主题研究：

```
/deep-research [您的研究主题]
```

### 使用 TikHub API

搜索相关 API：

```bash
# 按关键词搜索
python .claude/skills/tikhub-api-helper/api_searcher.py "用户资料"

# 按类别列出 API
python .claude/skills/tikhub-api-helper/api_searcher.py tag:TikTok-Web-API

# 获取详细的 API 信息
python .claude/skills/tikhub-api-helper/api_searcher.py detail:OPERATION_ID

# 发起 API 请求
python .claude/skills/tikhub-api-helper/api_client.py GET /api/v1/tiktok/web/fetch_user_profile "sec_user_id=USER_ID"
```

### 研究流程

1. **查询分类**: 系统确定研究是否需要深度优先、广度优先或直接的方法
2. **并行执行**: 多个子智能体同时研究不同方面
3. **工具选择**: 根据内容类型选择适当的工具
4. **数据收集**: 从多个来源收集信息
5. **综合**: 将发现整合到综合报告中
6. **引用**: 添加适当的引用以增强信任

## 研究方法论

该工具包采用系统性的研究方法：

1. **评估**: 分析查询并识别关键概念和所需数据点
2. **规划**: 确定最佳研究策略和子智能体部署
3. **执行**: 使用适当工具和方法进行并行研究
4. **验证**: 在多个来源之间交叉验证信息
5. **综合**: 将发现整合到综合、结构化的报告中
6. **引用**: 添加适当的归属和来源验证

## 最佳实践

- 使用具体、明确定义的研究查询以获得最佳结果
- 允许足够的时间让并行智能体完成综合研究
- 在多个来源之间验证关键信息
- 分析社交媒体影响时考虑文化背景
- 使用 TikHub API 获取结构化社交媒体数据，而不是网络抓取
